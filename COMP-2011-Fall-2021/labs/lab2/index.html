Game Description

We will be following the standard rules for the Mastermind game.

The game involves two parties, a codemaker and a code breaker. The codemaker sets a code consisting of 4 colored pins and it is the codebreakers objective to guess the code in 12 guesses or less, with the codemaker providing hints after each turn.

The code is made up of 4 pins. Each pin can be one of 6 possible colors: Yellow (1), Orange (2), Black (3), Green (4), Blue (5), or Purple (6). The codemaker randomly picks a color for each of the 4 positions and the code breaker must guess the correct color for each position within 12 turns.

After each turn, the codemaker gives hints to the code breaker.

  • For each poisition guessed, if the color is the same as the code, the codemaker shows a red pin as a response
  • If the color at the position is wrong, but the color appears at another position in the code, then the codemaker shows a white pin.
  • Each pin and position is considered once and only once when giving feedback.
  • Otherwise, no feedback is given

A maximum of 4 pins is given as feedback. The game ends once the codebreaker receives 4 red pins. For a video introduction to the game: https://www.youtube.com/watch?v=dMHxyulGrEk

Implementation

  • We assume the code is randomly selected by the codemaker. The colors are represented by the values 1-6
  • During the start of each round, we ask the codebreaker to guess the 4 pin code. The player enters 4 numbers between 1-6 and separated by a space. We assume the input is given correctly and do not perform input validation
  • The guess is evaluated and the feedback is automatically given by the codemaker. The total number of Red and White pins are shown.
  • If the codebreaker reaches the end of 12 turns, or the codebreaker guesses the code correctly, the game ends.
