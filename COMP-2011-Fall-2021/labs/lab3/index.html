COMP2011 Lab 3


Introduction

In this lab, you will practice function definition and calling by implementing filtering of a given list numbers. Please take skeleton code as your starting point and complete all the function definitions in lab3.cpp.

Tasks

The main goal of this lab is to implement several number analysis functions including digit occurrence counting, index finding and filtering.
Let's explain the functions with is an example.
Given a number, such as 12343456,

  1. The digit occurrence counting function is able to count how many times a digit, such as 3, appears in that number, which, in this case, should be 2 in the sense that dight 3 occurs twice.
  2. The index finding function is to return the position (index, starts with 0) of a digit at its n-th occurrence where n is specified by the user. For example, the first occurrence of the digit 1 is at index 0 and the second occurrence of the digit 4 is at index 5.
  3. The filtering function is to find out all digits that satisfy some criteria. Here we only consider occurrence criteria by printing all digits in a number that occur more than t times where t is specified by the user. For example, digits that occurs more than once are: 34. Also, the digits should be listed in an ascending order.

An interactive interface will be shown waiting for a user's instruction to proceed, which is able to support functions listed as below:

  • occurrence counting: calculating how many times a user-specified digit occurs in a user-specified number;
  • index finding: finding the index of the n-th occurrence (specified by the user as "order") of a user-specified digit in a user-specified number;
  • filtering by occurrence: printing all the digits in a user-specified number that occur more than a user-specified number of times.
  • quiting.

Grading Scheme

Each lab carries 3 points in total.

  • By attending a lab, a student gets 1 point for attendnace (but only if it is clear that the student does put in some effort).
  • By completing the lab exercise correctly and answering TAs' question (if there is a question) correctly, the student gets the remaining 2 more points.
If a student is unable to finish the lab and demonstrate to the TA by the end of the lab, they can only get the attendance point.
