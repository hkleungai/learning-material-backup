COMP2011 Lab 09

Introduction

This lab will help you better understand the use of linked lists, including creating, inversing and sorting a linked list.

A skeleton code has been provided to help you get started.

Description

Think back to your experience at primary schools when you were a kid. Teachers might ask all children to line up according to their heights. For example, tall children stand at the end of the line. This lab will use the scenario and invite you to use linked lists to simulate the process of lining up.

Tasks

Let us assume there are N kids in a class, and their heights are between [120, 160]. You are required to finish the following tasks:

  • Task 1: create an initial linked list consisting of N nodes with random heights, each presenting a kid.
  • Task 2: reverse the initial linked list.
  • Task 3: line up the kids according to their heights in an ascending order.

Hints

For task 2, as shown below, you can use two pointers, let them point to nodes one by one, and set cur->next=pre each time.

For task 3, you can use bubble sort or any other sort algorithms.
