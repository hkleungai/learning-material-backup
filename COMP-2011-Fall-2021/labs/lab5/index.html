COMP2011 Lab 5


Introduction

In this lab, you are going to implement a student grade recording system. The system supports functions including mergeFile, cleanFile, and analyzeFile. To finish the lab, you need to self-study C++ File I/O and C++ String.

Description

You will be given two dirty files which contain student names, grades, and some unwanted symbols, e.g., $%#@&*. Then you need to merge the two files to a new file, and clean the file. Finally, you need to find the two students who got the lowest/highest grade, respectively.

More concretely, the system contains the following four functions:

  • mergeFile: to merge two files to a new file.
  • cleanFile: to remove the unwanted symbols. Each line in the output file (clean file) will contain only one student's record, i.e., name and grade.
  • analyzeFile: to find the two students who got the lowest/highest grade, respectively. If there are two or more students who got the same lowest/highest grade, you only need to return the first-appeared records.
  • displayFile: to print out the content of the file. It is given in the skeleton codes.

Implementation

  • Start your coding with the package. It contains skeleton codes and sample files.
  • You don't need to change the function headers.
  • You can assume that there are 20 students at most, all names only consist of letters, all grades are integers, and there is at least one blank space or \n between names and grades.

Grading Scheme
