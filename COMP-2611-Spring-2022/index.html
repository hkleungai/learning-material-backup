Course Information


Course Description

Inner workings of modern digital computer systems and tradeoffs at the hardware-software interface. Topics include: instructions set design, memory systems, input-output systems, interrupts and exceptions, pipelining, performance and cost analysis, assembly language programming, and a survey of advanced architectures.


Learning Outcomes

Upon completion of the course, students are expected to be able to:

  • Understand the basic concepts of digital logic and build the small circuits involved in computer systems
  • Describe the interaction between software and hardware and instruction set architecture
  • Write and execute small programs of a few hundred lines in assembly language
  • Define the basic concepts of modern computer hardware, including datapath, control, memory and input/output
  • Describe the organizational paradigms that determine the capability and performance of computer systems

  • For Mac users, you can use Logisim without installing JAVA 6: Download and unzip the Logisim pkg, right-click icon and click show package contents, then follow contents->resources->java->logisim.jar , you can use this logisim.jar directly (Do not install JAVA 6).
  • There is a problem with Logisim on MacBook when storing a .circ file and loading it as a library. Some students ask this in the first Lab course. It may be a system problem. This issue can be solved in this way: After you stored the .circ file on the original system folders (e.g. Desktop, Downloads, Pictures), you will not find it on the path of Logisim Library. Please store the file in your personal folder, for example, my computer name is lee", and I have a self-created folder test, then I can store this .circ file in the /Users/lee/test/multiplexer.circ path and there is no loading problem from this path.
