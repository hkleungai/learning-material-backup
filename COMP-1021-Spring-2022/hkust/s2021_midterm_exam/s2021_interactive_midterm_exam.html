Question 12 (10 marks)



Students were allowed to copy the code for this question from the exam.




  Many people have died as a result of the coronavirus. The number of dead is so large it is hard to appreciate. Somebody makes the following program to try to help people understand how many have died. It shows a circle for each person who died. When the program is executed lots of circles are shown, then the user presses Enter, then the circles are cleared and more circles are shown. This pattern repeats until all the circles are shown. In this way one circle is shown for each person who died in that country.




import turtle

total_dead_in_country = 35254

turtle.color("orangered")
turtle.speed(0)
turtle.hideturtle()
turtle.up()

count = 0
while count < total_dead_in_country:
    from_user = input("You have seen " + str(count) + " circles - now press Enter")
    turtle.clear()
    for y in range(-200, 200, 20):
        for x in range(-200, 200, 20):
            if count < total_dead_in_country:
                turtle.goto(x, y)
                turtle.dot(10) # This works even if the pen is off the page
                count = count+1
print("You have seen " + str(count) + " circles")
    
turtle.done()





  You are using the program to illustrate how many people have died in a particular country. In that country, 35254 people died.





The question is: how many times does the user need to press Enter in order to see all the people who died in that country? For your answer, simply enter an integer.





  Note: you can see input() is used. In this question we don't care if the user actually enters some text. We just want them to press Enter (no need to enter any text). If they do enter some text it is ignored.





Note: on some keyboards the Enter key is called the Return key. They are the same thing.
