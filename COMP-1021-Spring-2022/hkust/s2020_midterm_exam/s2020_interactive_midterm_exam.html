Question 5 (8 pts)

Here is a Python program.

After you start the program, what series of text do you need to enter into the program in order to find the exit (which would then trigger the end of the program)?

row = 2
col = 1
finished = False
locations = [ ["Trees",     "Enemies Tower",    "Wall"],
              ["Top Lane",  "Treasure Cave",    "Middle Lane" ],
              ["Tower",	    "Jungle",           "Monster Cave"] ]

while not finished:
    print("You are in a", locations[row][col])

    direction = input("Which way? (N/S/E/W) ")
    if direction == "N":
        if row == 0:
            print("Sorry, you can't go that way!")
        else:
            row = row - 1
    elif direction == "S":
        if row == len(locations)-1:
            print("Sorry, you can't go that way!")
        else:
            row = row + 1
    elif direction == "W":
        if col == 0:
            print("Sorry, you can't go that way!")
        else:
            col = col - 1
    elif direction == "E":
        if col == len(locations[0])-1:
            print("Sorry, you can't go that way!")
        else:
            col = col + 1

    if col == len(locations[0]) and row == len(locations):
        print("You have found the exit!")
        finished = True

If you think that it is not possible to reach the exit (which would then trigger the end of the program), write NOT POSSIBLE in the answer.

Please enter your answer in CAPITAL LETTERS.

Your answer:

Correct answer: NOT POSSIBLE
