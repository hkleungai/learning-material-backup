'Net worth' means the sum of everything a person owns. For example, if you have HK$3,000 in your bank account, HK$1,000 in cash, a laptop worth HK$5,000 and a phone worth HK$1,000, then your net worth is HK$10,000.





Using 1 rectangle to represent HK$1,000 you could visualize your net worth like this:








Several people have developed IT companies and become very wealthy. The net worth of some people is so large it is hard to appreciate. Somebody creates the following program to try to help them understand the net worth of a particular person. It shows a rectangle for each HK$1,000 of the person's net worth. When the program is executed lots of rectangles are shown, then the user presses Enter, then the rectangles are cleared and more rectangles are shown. One rectangle represents HK$1,000. This pattern repeats until all the rectangles are shown. In this way the complete net worth of the person is shown.




import turtle
turtle.color("orange")
turtle.hideturtle()
turtle.up()

# This variable represents the net worth in thousands (HK$1,000)
wealth_in_thousand_hkd = 713700000
count = 0
while count < wealth_in_thousand_hkd:
    from_user = input("You have seen" + str(count) + \
                      "rectangles - now press Enter")
    turtle.clear()
    for y in range(-210, 210, 25):
        for x in range(-210, 210, 25):
            if count < wealth_in_thousand_hkd:
                turtle.goto(x, y)
                turtle.begin_fill()
                turtle.forward(15)
                turtle.left(90)
                turtle.forward(10)
                turtle.left(90)
                turtle.forward(15)
                turtle.left(90)
                turtle.forward(10)
                turtle.left(90)
                turtle.end_fill()
                count = count + 1
print("You have seen" + str(count) + "rectangles")
turtle.done()

  Here is a program that changes some text into a 'secret message', which does not resemble its original content. Using the secret message, you can send sensitive information to other people without revealing the actual content when it is transmitted. If you want to, you can copy the code from this web page, fix the indentation, and use it in your Python editor.




# This program changes a piece of text into a secret message.

# The text to be changed can only contain the following letters
letters = "abcdefghijklmnopqrstuvwxyz"

# The text to be changed
input_text = "crazyworld"

# The secret code that is used in the process
code = "159145"

# The variable to store the secret message
result = ""

for i in range(len(input_text)):
    # Get the current letter
    letter = input_text[i]

    # Find the location of the letter and the offset to its 'secret' version 
    index = letters.index(letter)
    offset = int(code[i % len(code)])

    # Add the 'secret' letter to the result
    result = result + letters[(index + offset) % len(letters)]

# Show the result
print(result)





  In the above example, the text 'crazyworld' is changed using the secret code '159145'. The result is the secret message 'dwjacbpwue'. This secret message can then be sent to other people. They can use a decoding program (which is not given in this question) and the same secret code (159145) to change the secret message back into its original content.





  Here is a brief description of how the program works.





  At the start of the program, you have a list, letters, containing the letters that you can use in the input text. You are also given the input text and the secret code in the variables input_text and code respectively.





  Then, for each letter in the input text, the program changes the letter to a secret version of itself. The secret version is another letter to the right of the original letter inside letters. How far the secret version is to the right of the letter is determined by the secret code and the current location of the letter. This is calculated and stored in the variable offset.





  When finding the secret version of the letter, it may be located way over the right hand side of letters. In this case, the program looks for the secret version back to the left hand side of letters, as indicated by the use of the code % len(letters).
