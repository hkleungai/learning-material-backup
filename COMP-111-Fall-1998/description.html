COMP 111 (Software Tools)

Course vector: [2-0-2:3]
Exclusion: COMP 211
Pre-requisites: COMP 102 or COMP 104

Course Objectives:

  • learn a large set of specific, particularly useful UNIX software tools
  • learn how to write unix software tools
  • learn the principles of what makes a good software tool

Major topics: 26 lectures

  • Getting started with Unix [2 lectures]
  • Basic Internet [3 lectures]
  • Basic X-windows [2 lectures]
  • More Unix utilities [2 lectures]
  • Unix file structure [3 lectures]
  • Basic shell [2 lectures]
  • Bourne shell programming [8 lectures]
  • Regular expressions [1 lecture]
  • Sed [1 lecture]
  • Awk [2 lectures]

Recommended implementation

Text Book(s):
Unix System V: A Practical Guide; Mark G. Sobell, The Benjamin/Cummings Publishing, 3rd Edition, 1995.

Reference Book(s):
(optional) Unix Power Tools, Jerry Peek, Tim O'Reilly and Mike Loukides, Bantam, 1993 (a new edition is coming out this year)

Workload and Grading Scheme:

12 Labs (10%)
3 Homework Assignments (10% each)
1 mid term examination (25%)
1 final examination (35%)

Lectures: (50 minutes per lecture)
 
 
Lecture Hours Topics
2 Getting started with Unix: log in/out; password; special control keys; simple text editing; online help; printing; simple file manipulation (ls, cat, more, less, rm, cp, mv); simple e-mail
3 Basic Internet: basic network concept; basic Internet concept (domain name, IP address); basic Internet tools (ftp, finger, telnet, rlogin, usenet, WWW, gopher)
2 Basic X-Windows: basic concept of client-server; basic concept of windows; using the X-windows system (application clients, window managers, X servers) 
2 More Unix utilities: Unix shells; who, write, talk, apropos, echo, date, diff, head, tail,
grep, sort, uniq; input/output redirecion
3 Unix file structures: creation (mkdir); removal (rmdir); movement (mv); categorization
(path, special directory); security (ls -ld, chmod), links (hard, symbolic)
2 Shell basics: what a shell does and how it does it; standard input/output/error; filter; pipe; background/foreground jobs and job control; finding information on & controlling processes
8 Shell programming (using the bourne shell): shell scripts; shell variables (user-defined,
environment); command substitution; flow control (if, for, while, until, case);
1 Regular expressions: ., *, ^, $, [], [^], \c, illustrated with grep 
1 Sed
2 Awk

Laboratory exercises:
Lab 1: to practise the following commands: login, change password, simple editing, cp, ls, cat, head, tail, man, mv, mail, rm
Lab 2: to practice the following commands: save a newsgroup message, simple editing, finger, talk, ftp
Lab 3: to learn how to construct personal homepage
Lab 4: to learn more about X-windows
Lab 5: to practise on file management commands: mkdir, ls, chmod to practise on shell basics, such as |, >, <, tee, grep
Lab 6: to discuss Homework 1
Lab 7: to learn how to assemble a set of commands to solve a specific problem
Lab 8: to discuss the mid term exam
Lab 9: to learn how to write a simple shell script
Lab 10: to write one more shell script
Lab 11: to practise on regular expression and Sed
Lab 12: to work on Homework 3

Home assignments:
Assignment 1: to learn how to use a set of basic software tools in problem solving
Assignment 2: to write shell programs
Assignment 3: to write Sed/Awk programs
