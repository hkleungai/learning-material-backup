Printing lecture notes

You will be shown how to access the COMP111 lecture notes, and how to print them.

Note that you have a fixed printing quota. When you run out of quota, you cannot print any more pages! So it is a good idea to save paper by printing two or four pages on every side of paper. To do this you need to :

  1. Using Netscape, save the file which contains the lecture notes to your directory.
  2. Use the mpage command to create a new file which has multiple pages on each side (for example,
    mpage -4 lecture.ps > newfile.ps
    will produce four pages on each side).
  3. Optionally, you could use the program ghostview to look at the new file. To do this, just type
    ghostview newfile.ps
  4. Print this new file using lpr (for example,
    lpr -Pcll3 newfile.ps
    will send the new file to the printer).

See the documents on schedule for quota reset and on how to checking print quota.

    3. pico is a simple text editor. 
The command "ls" lists directory contents and file attributes. 
The command "cat" concatenates and displays file contents. 
The command "rm" removes a file. 
The command "cp" copies source file(s) to target file(s).
