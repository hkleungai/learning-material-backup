Exam Seating

Due to the large number of students in the class,  the midterms will be held in multiple venues at the same time.  You will be assigned the venue in which you should write the exam. A few days before each exam is held,  we will post the venue assignments to the class web page.

Study Tips for the exam

  • Look at old exams
  • Make sure that you understand solutions to all of the homeworks and supplementary questions.
    • We will not ask the same questions on the exams but we might ask some similar ones
  • Practice writing explanations of solutions.
    • We will be deducting points for solutions without justifications
  • Try practicing on problems in back of the book chapters.
    • If you have trouble, go see the TAs (any TA, not just yours). They know answers to all of these problems.
  • For copyright reasons Homework Solutions can only be accessed from WITHIN the UST internet domain.
    • If you live or study off-campus, make sure that you download the solutions before you leave campus.
