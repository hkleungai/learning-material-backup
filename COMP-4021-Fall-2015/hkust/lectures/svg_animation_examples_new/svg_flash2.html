Flash Tweening - Two Types
  1. Motion Tween. The source frame and destination frame contain the same object. Only size, position, orientation and visibility can be changed during the transformation. The object must be grouped or contained in a symbol
  2. Shape Tween. It transforms graphic from any shape to any other shape. All objects must be ungrouped.
