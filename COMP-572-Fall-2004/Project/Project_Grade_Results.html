Project Grades

The project grades are now available here. Please note the following

  1. Projects were graded on  a scale of 0-100. 
    Your final grade on the project is listed in the final column (Project) of the table.
  2. Team members on the same project received the same grades
  3. 30 points  (Proj Doc) were given for documentation and program style;
    70 points (Proj Prog)  for correctness
  4. The 70 correctness points were awarded after running the programs on various standard test cases. Points were deducted for each test case on which the program did not give the proper answer. We then asked the authors of penalized programs to correct their code and reran them again on the test cases. Points were again deducted for all test cases yielding improper answers .  We sent the still-penalized programs back for fixing a 2nd time.  On return, all programs were working on all test cases so no more points were deducted.
