COMP 104: LAB 1

  1. Click Start button (lower left corner) and Choose Run.

  2. Type, for instance, notepad c:\documents and settings\eg_XXX\desktop\hello.cpp to create an empty C++ program file.

  3. Cut and paste the following C++ program into your notepad:

            #include <iostream>
            using namespace std;
            int main(){
                cout << "hello world!" << endl;
                return 0;
            }

  4. Save (by press CTRL-S) the file. Exit notepad.

  5. Double click your C++ program. Microsoft Visual C++ 6.0 will be launched.

  6. Compile your C++ program. (Guess what button you should press.)

  7. Run your C++ program. (Guess what button you should press.)
