Setup

While there are other ways the easiest is to unzip WPy-3662.zip (Tensorflow 1.10.0) (768M) or WPy-3662-new.zip (Tensorflow 2.0.0 & PyTorch 1.7.1) (1.5 Gb) in your local Window machine. If you enounter an error of "missing asmjit.dll," please install https://aka.ms/vs/16/release/vc_redist.x64.exe. You can also use Google Colab to complete the assignment. Please submit your Jupyter Notebook with the results.

All the assignments (starting codes and our solutions) are tested in Lab 4 (Room 4210, lift 19), which has GPU support and is open 24 hours. All local files saved elsewhere will be removed when the lab machines are rebooted every 24 hours at 5am, so please back up your code and data each time you use Lab 4. You need a CSD account to use Lab 4.

For Assignment 1, you can use the provided WPy-3662.zip or WPy-3662-new.zip in Windows, or you can build your own environment in Mac and Linux. After unzipping the file, you can use "WinPython Command Prompt.exe" or "Jupyter Notebook.exe" to write your code (py or ipynb files). To install libraries such as Tensorflow (CPU version), you can run "pip install tensorflow==1.10.0" in "WinPython Command Prompt.exe". You can also use Tensorflow or PyTorch for this assignment.

You can write your code in Python or Jupyter Notebook. You may use Tensorflow, PyTorch, or libraries at your convenience. There is a good tutorial on Tensorflow: https://github.com/easy-tensorflow/easy-tensorflow

In case you are not familiar with Python, you can go through http://cs231n.github.io/python-numpy-tutorial/

Please submit your assignment on Canvas.
