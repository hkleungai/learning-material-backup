Question 3: 15 Coins

In this lab you will program a simple coin game where the user plays against the computer. The game begins with 15 coins on the table. Each turn a player can take 1, 2 or 3 coins away from the pile. The player who takes the last coin loses.

Assume the computer always begins first. Your task is to design a way to make sure computer always wins the game, no matter what the user chooses.

Here is a sample game:

        Starting with 15 coins
        Computer takes 2, 13 coins remaining
        How many coins do you want: 3
        User takes 3, 10 coins remaining
        Computer takes 1, 9 coins remaining
        How many coins do you want: 3
        User takes 3, 6 coins remaining
        Computer takes 1, 5 coins remaining
        How many coins do you want: 3
        User takes 3, 2 coins remaining
        Computer takes 1, 1 coins remaining
        User is left with final coin, user loses

(Hint: Getting the right idea to solve this problem is a bit tricky. Implementing the idea as a program is not difficult.)
