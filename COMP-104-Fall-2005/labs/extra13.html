Question 1: Square Struct

Implement and test a Square struct with the following declaration

struct Square{
        int length;             // length of side
        char borderChar;        // the border character
        char fillChar;          // the fill character
};

When you print a square and length = 4, borderChar='X', and fillChar=' ', your program should print:

XXXX
X  X
X  X
XXXX

When you print a square and length = 5, borderChar='O', and fillChar='o', your program should print:

OOOOO
OoooO
OoooO
OoooO
OOOOO
