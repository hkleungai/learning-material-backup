Printing

To load these files, click on the blue coloured words. If the program you are using (probably `Netscape') has been configured correctly, the postscript previewer program will automatically start and the file will be loaded into the program. Then, to print the file, click on the `Print' option, which is on the `File' menu item.

In order to save papers, you may compress several pages into one sheet of paper. If you are using a Unix-based system, there is a command "mpage" that allows you to do that. The required steps are:

Step 1: save your postcript file

Step 2: mpage -2 -P{printer name} {your postscript file}

Note:

Please replace {printer name}, inclusive of the curly brackets, by the name of a postscript printer that you want to use. You should also replace {your postscript file}, also inclusive of the curly brackets, by your actual filename.

The flag "-2" means compressing two pages into 1 page. If you want to compress 4 pages into 1 page, the command should be:

mpage -4 -P{printer name} {your postscript file}

For a more detailed description of "mpage", please do a "man mpage" command to consult the online manual.

To determine what printers are available and where they are, please do a "man printers" command.
