Basic Instructions in Writing Visual C++ Programs

Start up Visual C++

  •      Click the "Start" button on the bottom-left corner of the screen.
  •      Select "Programs" from the pop-up menu.
  •      Select "Microsoft Visual Studio 6.0" -- a submenu will pop-up.
  •      Select "Microsoft Visual C++ 6.0" from the submenu (Visual C++ will start running).
Create a new C++ file
  •      Click "New" from the "File" menu.
  •      Choose the "Files" tab.
  •      On the left side of the window, choose "C++ Source File".
  •      On the right side of the window, enter the file name "lab1".
  •      Also on the right side of the window, enter the location "C:\Documents and Settings\login" (where login is your login name).
  •      Click the "OK" button.
A new blank file "lab1.cpp" will be created, and a new window for that file will be displayed. Copy and paste the "Hello World" program into the file. Now save your program:
  •  Click "Save" from the "File" menu.
Compile, link, and run your program

Now you are ready to compile your program. If there are any syntax errors, error messages will be displayed in a window at the bottom of the screen. You should correct all the errors.

  •  To compile your program, click "Compile" from the "Build" menu. (Hit the "Yes" button to create a default project workspace.)
  •  To link your program, click "Build" from the "Build" menu.
  •  To run your program, click "Execute" from the "Build" menu.
A DOS prompt window will be displayed, and your program will start running. Hit any key and the DOS window will disappear.

Save your file to floppy disk

  •  Click "Save As..." from the "File" menu, and save your file in A:\lab1.cpp.
 Use Windows Explorer to delete C:\Documents and Settings\login\lab1.cpp, and empty the recycle bin. Otherwise your work may be lost or stolen by others.
