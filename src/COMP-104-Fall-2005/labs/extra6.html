Question 1: Tic Tac Toe

Write a program that allows the user to play the computer in the game "tic tac toe". The computer should go first and play "O". The user should go second and play "X". For the computer strategy, implement the following algorithm:

        1) If the computer can win on the next move, do it,
        2) If the user can win on the next move, block it
        3) Pick a random unoccupied square.

Make the user re-enter a position if it is illegal (already taken or off the board). Print out the board after each move, and detect and report when either the user or computer wins the game.
