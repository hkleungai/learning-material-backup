17/02/2016

Accessing your web application folder on the M drive from the Virtual Desktop

As the Virtual Desktop is not hosted on the CSE department servers (it is hosted on ITSC servers), it is not possible to access the M drive directly from the Virtual Desktop. You have two options for accessing your web application if you have saved it to the M drive.

  1. DO NOT save your web application to the M drive.
    Instead, save it to a USB drive. You can then upload your web application onto the Virtual Desktop from your USB drive.

  2. Use SFTP to access your web application folder on the M drive.
    From the Virtual Desktop, login to one of the Lab 2 Unix computers (i.e., csl2wk01.cse.ust.hk to csl2wk41.cse.ust.hk) using SFTP and then upload your web application folder from the ~/DOS folder on the Unix computer.

The SFTP program to use on the Virtual Desktop is "FileZilla", which can be found in the folder "FileZilla FTP Client" in "All Programs" in the Start menu.

NOTE: You need to enter "sftp://csl2wk??.cse.ust.hk" in the "Host:" field where you replace ?? by a number ranging from 01 to 41.
