COMP1022Q Introduction to Computing with Excel VBA Course Rubrics

Course Outcomes

Learning Outcome 1:
Manipulate and visualize data using Microsoft Excel.
Learning Outcome 2:
Program using the VBA language within Microsoft Excel to build effective solutions to problems.

Levels of Achievement

Level of Achievement
Learning Outcome 1
Learning Outcome 2
Exemplary
The student is able to proficiently manipulate and visualize data within Microsoft Excel.
Given a problem definition, the student is able to proficiently use the VBA language within Microsoft Excel to solve the problem. The program can return the correct result on all data sets.
Competent
The student is able to manipulate and visualize data within Microsoft Excel satisfactorily, but sometimes makes mistakes on certain difficult tasks.
Given a problem definition, the student is able to use the VBA language within Microsoft Excel to solve the problem satisfactorily. The program can return the correct result on almost all data sets, but may miss some special or boundary cases.
Needs Work
The student is barely able to manipulate or visualize data within Microsoft Excel, and often makes mistakes.
Given a problem definition, the student is barely able to use the VBA language within Microsoft Excel to solve the problem. The program often returns incorrect results.
Unsatisfactory
The student is not able to manipulate or visualize data within Microsoft Excel.
Given a problem definition, the student is not able to solve it using the VBA language.
