COMP1022Q Introduction to Computing with Excel VBA Notes Examples - Lookup Techniques

  • 01_1022q_grade_performance_using_vlookup.xlsx [from page 5 of the notes]
    • This example demonstrates using the VLOOKUP cell function to convert letter grades to descriptions
  • 02_1022q_grade_performance_using_hlookup.xlsx [from page 10 of the notes]
    • This example is almost the same as the previous example except that the HLOOKUP cell function is used this time
  • 03_1022q_grading_with_approximate_matching.xlsx [from page 16 of the notes]
    • This example demonstrates using approximate matching to convert exam scores to letter grades
  • 04_1022q_grading_without_approximate_matching.xlsx [from page 18 of the notes]
    • This example is the same as the previous example except that the lookup has been done without approximate matching, which results in #N/A errors
  • 05_1022q_coffee_comparison.xlsx [from page 21 of the notes]
    • This example demonstrates the use of IF and ISNA to handle the 'not available' error returned by the lookup function
  • 06_1022q_coffee_comparison_ifna.xlsx [from page 25 of the notes]
    • To make this example the previous example, example 05, was converted to use IFNA. The result is a much shorter formula.
