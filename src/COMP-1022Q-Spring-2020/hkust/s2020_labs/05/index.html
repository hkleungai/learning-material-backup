2. Write the VBA Code to Move the Pieces Randomly

  • Now you can return to Excel and open the VBA Editor from the Developer tab

2.1. Locate the LoadJigsawPieces() macro

  • We will use a macro called LoadJigsawPieces() to create the jigsaw pieces
  • You can run the macro by pressing Ctrl-Shift-L
  • You can find and edit the macro from the macro window
  • Locate the macro LoadJigsawPieces() provided in the starting code 
    Sub LoadJigsawPieces()

    'This sub-routine will load the jigsaw piece into the worksheet

    'You should write your code below

End Sub
  • You can set up two variables to define the total number of columns and rows, like this: 
    Dim TotalRows As Integer
Dim TotalColumns As Integer

TotalRows = 3
TotalColumns = 3

2.2. Set up the nested loops

  • Here is the structure of our two levels of loops: 
    For loop going through the rows
    For loop going through the columns
        Generate a random position for the shape
        Create a shape at this position
        Prepare the name of the shape
        Prepare the name of the image file that goes with this shape
        Load the image file into the shape
    Next column
Next row
  • You can prepare the shell of the outside for loop first: 
    For Row = 1 To TotalRows ' Go through the rows one by one
    ' We will prepare the loop contents later
Next Row
  • Next, you should add the inner loop: 
    For Row = 1 To TotalRows ' Go through the rows one by one
    For Column = 1 To TotalColumns ' Go through the columns one by one
      ' We will prepare the loop contents later
    Next Column
Next Row
  • Remember, you need to use Dim to define Row and Column at the top of this subroutine

2.3. Contents of the loops

2.3.1. Generate a random position

  • The jigsaw pieces will be put at random positions on the screen at start of the game
  • To generate a random number, you have to first run this command to ensure you have different random numbers every time: 
    Randomize
  • Then, you can generate a random x position from 0 to 800 with this command: 
    X = Int(Rnd() * 800)
    • Int() converts the result into integer
    • Remember to 'Dim' the variables
  • Please prepare a Y position similarly
  • To check if these variables are being prepared properly, we can use the MsgBox as a debugging tool, for example: 
    MsgBox "Row = " & Row
  • You can now try this subroutine by closing and reopening your Excel file, or simply by choosing Run in the VBA Editor toolbar
  • After you confirm that the variables Row, Column, X, and Y are working properly, you should remove the MsgBox
    • Or you may convert the command into a comment by adding the symbol ' in front, so that the command will be ignored by VBA

2.3.2. Create a shape at this position

  • You can create a rectangle shape with VBA using this command: 
    ActiveSheet.Shapes.AddShape( msoShapeRectangle, left, top, width, height )
  • You need to fill in the correct parameters:
    • type: The type of shape we use this time is msoShapeRectangle
      • msoShapeRectangle is just a special variable which takes the value of 1
        • For example, this will show you 1: 
          MsgBox msoShapeRectangle
      • You can find more shape names here:
        https://msdn.microsoft.com/en-us/VBA/Office-Shared-VBA/articles/msoautoshapetype-enumeration-office
    • left / top: You will use the two variables X and Y that you have prepared in section 2.3.1
      • The left is the x position of the shape (the top left corner)
      • The top is the y position of the shape (the top left corner)
    • width / height: These are the width and height of each jigsaw piece image
    • To check out the width and height of the jigsaw piece images, you can right click on the file and choose Properties (on Windows) or Get Info (on Mac), for example:

      File properties on Windows


      File info on Mac
      • The width of this image is 500, and the height is 282
  • In this lab, you need to put the generated shape into a variable for further adjustments: 
    Set ThisShape = ActiveSheet.Shapes.AddShape(  . . .  )
    • This variable ThisShape also needs to be created before it is used, as a Shape type

2.3.3. Prepare the names used by this jigsaw piece

  • You need to prepare the name of this shape, to be used in the checking stage later: 
    ThisShape.Name = Column & "-" & Row   ' e.g. Row=2, Column=3 => "3-2"
  • Remember you can always use MsgBox to check what is in a variable e.g. 
    MsgBox ThisShape.Name   ' This shows the generated shape name
    • Do not forget to remove this MsgBox after you confirm that the variable is working fine
  • Then, you can set up the image file name to be loaded, using the Row and Column variables, according to your actual file names:
  • The file name with variables can be built by: 
    ThisFilename = front part of the file name & Row & "-" & Column & ".jpg"
    • In our example, the front part is "image-"
      • Check that you named all your files properly!
    • If you did not change the file type, the image should be a jpg file (a common image file format)
    • ThisFilename should be a String variable
  • We build our file name using 5 pieces of text here, connected by the & operator
    • The Row and Column are the variables we used earlier, for example 2 and 3 respectively
      • Remember we will use a 3*3 structure for our jigsaw in this lab
    • In our example, the variable ThisFilename will be image-2-3.jpg

2.3.4. Load the image file

  • Finally, you can load the image file into the shape: 
    ThisShape.Fill.UserPicture ThisWorkbook.Path & "/" & the filename variable you created
    • You just need to fill in the variable name you used in section 2.3.3
  • This command works fine for both Windows and Mac machines

3. Check if the Jigsaw is Correct

3.1 Overview of checking the jigsaw

  • To allow players to check if the jigsaw pieces are arranged correctly, a subroutine CheckJigsawPieces() has been built for you
  • Although the code is already given to you, you need to read and understand it
  • The code uses the For Each loop, which will be introduced later in the lectures
  • The For Each loop gets each shape from the collection of all shapes and repeats the loop content
  • The basic idea of the subroutine works like this: 
    First assume jigsaw is correct

A For Each loop goes through every shape
    Obtain row/column data from shape name using text handling functions Left() and Right()
    Obtain shape position using .Left and .Top properties

    Another For Each loop goes through every shape in worksheet for comparison
        Obtain row/column data of compared shape
        Obtain compared shape position

    If the position of the current shape violates the row/column data when comparing with another shape:
        Jigsaw is wrong

After the loop has finished, show an appropriate message
  • Here you can see how the jigsaw pieces could be aligned in a 2×2 jigsaw:
  • We need to check, for example
    • The jigsaw piece for row 1 column 2 (i.e. ThisShape) should NOT be below the piece for row 2 column 2
    • The jigsaw piece for row 1 column 2 (i.e. ThisShape) should NOT be on the left of the piece for row 1 column 1
  • With a good understanding, you should create the shortcut key to run this as a macro using Ctrl-Shift-C
