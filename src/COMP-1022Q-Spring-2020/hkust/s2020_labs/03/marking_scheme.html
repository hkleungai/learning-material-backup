COMP1022Q Lab 3 Marking Scheme

Marking Scheme

  1. Creating an integer random number (15 marks)
    • VBA function Randomize is used to initialize the random number generator (5 marks)
    • The variable Target is set by an appropriate random number with a range of 1 to 100 (including 1 and 100) (5 marks)
    • Show a message box such as "I am thinking of a number..." (5 marks)
  2. In the loop, get the input from the player (5 marks)
    • Show an input box to ask for an input number (5 marks)
  3. In the loop, checking the player's guess (25 marks)
    • Make sure the guessed number is in the range of 1 to 100 (5 marks)
    • Show an appropriate error message if the range is lower than 1 or higher than 100 (5 marks)
    • If the guessed number is higher than the player's random number, tell the player to try again with a lower number (5 marks)
    • If the guessed number is lower than the player's random number, tell the player to try again with a higher number (5 marks)
    • If none of the above conditions is correct, the guessed number will be the computer's random number, tell the player he/she has a correct guess (5 marks)
  4. Counting the number of guesses (15 marks)
    • The variable NumberOfGuesses is initialized as 0 (5 marks)
    • The variable NumberOfGuesses is increased by 1 every time a guess is made (5 marks)
    • Show the total number of guesses when the game is over (5 marks)
  5. Showing the game history in Excel cells (30 marks)
    • The variable CurrentRow is initialized to 2 (5 marks)
    • For each guess, show the guess number in an appropriate row in column A (5 marks)
    • For each guess, show the response message in an appropriate row in column B (10 marks)
    • The CurrentRow variable should increase by 1 every time after writing (10 marks)
  6. Clearing the worksheet before each game (10 marks)
    • Starting from row 2, clear a suitable number of row in columns A and B in order to remove the past game history
      • In the lab, Range("A2:B102").ClearContents is recommended
  7. Enter your name and student ID number somewhere at/near the top of column C. In the lab, Cell C1 is recommended
    • This is your statement that the work is yours
