1. Use Cell Formulas to Calculate the Ball Path in the Calculations Worksheet

1.1 Things We Need to Calculate the Ball Positions

  • We don't actually show a ball, we just show the path of the ball
  • This video shows the changes in the Calculations worksheet when you change the Gravity and Speed values (there's no sound):
    • A direct link to the video is here
  • The calculations which work out the position of the ball are done in the Calculations worksheet, using cell formulas
  • The formulas need these two inputs from the Game worksheet to calculate the ball positions:
    1. A cell called Gravity, located at B7 of the Game worksheet
      • This value doesn't exactly mean gravity
      • It is just a number which 'pulls' the ball downwards
      • The value given to you in the starter Excel file is 0.8
      • The value can be changed by the user
    2. A cell called Speed, located at B8 of the Game worksheet
      • The value given to you in the starter Excel file is 5
      • The value can be changed by the user

1.2 Calculate Vertical Positions (Y) using the Horizontal Positions (X), Gravity, and Speed

  • This image shows you the area in the Calculations worksheet where we will be working in the following stages:

    • Click on the image to see its original size
  • This image shows the general idea behind the columns:

    • Click on the image to see its original size
  • Calculating the column values
    • Go to the Calculations worksheet
    • To display the ball path properly, we need to map the (X,Y) coordinates to Excel Rows and Columns
    • Mapping the horizontal position (X) is trivial. The Column value should always be the same as X
    • So the following formula should be applied to Cell C2: 
      =A2
    • Copy down the formula to the range in C2:C101
  • Before writing the formula for the vertical position (Y), let's understand what we are going to calculate
    • We can apply the following Excel formula in B2, the first cell of the column of vertical position (Y) 
      =Speed * A2 - 0.5 * Gravity * A2 * A2
    • For Speed * A2, you can treat it as an upward force
    • For - 0.5 * Gravity * A2 * A2 , you can treat it as a downward force
    • Don't worry too much about what the formula exactly does. This course is not a Physics course!
    • Speed and Gravity are the cells mentioned in the last section
    • We can use those names in any cell formula, which is what we have done here
    • Copy the formula to all the cells below (cells B2:B101)

1.3 Calculate Row and Column values using the (X, Y) coordinates

  • Calculating the row values
    • Calculating row values is a bit tricky. You need to first locate the starting position of the ball
    • Select the Game worksheet
      • You can see a black line between row 40 and 41

        • Click on the image to see its original size
      • That is the ground of the throwing ball game
    • In this game, the ball always being thrown from row 40
    • Select the Calculations worksheet by clicking the "Calculations" tab at the bottom
    • The following formula should be applied to Cell D2: 
      =INT(40-B2)
    • The INT function in the above formula is used to convert a number with a decimal place to an integer
    • Let's use Row 2 as an example:
      • 4.6 is the value calculated by the formula in section 1.2 above
      • 40 - 4.6 is equal to 35.4
      • If we apply the cell INT function, the part after the decimal place will be discarded (there's no rounding)
      • The result will become 35
    • Copy the formula to the range in D2:D101

2.2 Initialise Every Cell to White

  • First, let's set the background of every cell to white
    • We would like to clear all existing ball paths and any previous target on the screen
  • In VBA, we can type in the following code to achieve the result: 
    Cells.Interior.ColorIndex = 2      '2 = white
    • Cells, without a row and column, refer to all cells in the current Excel worksheet
  • In Excel, the colour index of white is 2. A list of colours can be found here
  • To test the current version of the macro
    • Select the Game worksheet by clicking the "Game" tab at the bottom
    • Press Ctrl-Shift-T to trigger this macro
    • All cells should be changed back to the white colour

2.3 Calculate a Random Position for the Target Cell

  • Next, we need to calculate a random position for the target cell
  • You should store the position in Row and Column respectively
  • Both of them should be a random value in the range from 1 to 40 (inclusive)
  • In VBA, we can type in the following programming statements: 
     Column = Int(Rnd() * 40) + 1
 Row = Int(Rnd() * 40) + 1
  • Explanation
    • Rnd()   The Rnd function generates a random number that is greater than or equal to 0 and less than 1
    • Rnd()*40   Multiplying this value with 40 will give us a number that is greater than or equal to 0 and less than 40
    • Int(Rnd()*40)   Applying a Int function will discard the decimal places, which gives us an integer from 0 to 39 (inclusive)
    • Int(Rnd()*40)+1   Adding 1 will give us a random integer from 1 to 40 (inclusive)
    • Generating a random number has been discussed in the lectures

2.4 Modify the Background Colour of the Target Cell

  • Next, we need to fill in the target cell using red colour
    • A list of colours can be found here
  • As you can see in the list, the colour index of red is 3
  • In VBA, we can type in the following programming statement: 
    Cells(Row, Column).Interior.ColorIndex = 3      '3 = red
  • Row and Column are 2 integers calculated in the previous step
  • To test the current version of this macro
