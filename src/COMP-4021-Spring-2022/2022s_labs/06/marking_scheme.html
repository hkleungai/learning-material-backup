COMP4021 Internet Computing > Lab 5/6 - Marking Scheme

Part 1 - User Registration (20 marks)

  • Validating user input data
    • Data validation is correct for all of the cases below - 5 marks
      • Username, avatar, name cannot be empty
      • Username can only contains underscore, letters or numbers
      • Username cannot exist in the current user database
    • Appropriate errors are shown for each of the above cases - 5 marks
  • Adding a new user
    • The password is stored as a hash - 5 marks
    • New user data is added to users.json - 5 marks

Part 2 - User Sign-In (15 marks)

  • Signing in the system
    • The server validates the username and password correctly - 5 marks
    • A newly added user can sign in the system - 5 marks
  • Maintaining the session
    • A signed-in user does not need to sign in again after reloading the page - 5 marks

Part 3 - User Sign-Out (5 marks)

  • Signing out from the system
    • The user can sign out from the system by clicking on the sign-out button - 5 marks

Part 4 - The Online User List (20 marks)

  • Showing the online users
    • After signing in, all other currently signed-in users are shown in the online user list - 10 marks
    • The online user list is immediately updated when a new user signs in the system - 5 marks
    • The online user list is immediately updated when a user signs out from the system - 5 marks

Part 5 - The Chatroom (20 marks)

  • Managing the chatroom content
    • After signing in, the initial chatroom content is shown correctly - 5 marks
    • After posting a new message, the message is added to chatroom.json correctly - 10 marks
    • After posting a new message, the message is immediately shown in all users' chatroom - 5 marks

Part 6 - Improvement (20 marks)

  • Showing a 'typing' message
    • When a user presses any key on the new message input box, a 'typing' message is shown in all other users' chatroom area - 10 marks
    • The 'typing' message contains the name of the user who is currently typing on the input box - 5 marks
    • The 'typing' message disappears automatically after a certain time, for example 3 seconds - 5 marks
