Q 1.12:  The correct answer should be:  grep -v "[abc]" names instead of grep -v "[a,b,c]" names, since the latter will take "," as a character also.

Q 1.16: tar cf - . means to tape-archive the current directory ".", and then to print the result on screen (this is what "-" stands for).  Thus, if you pipe it later to gzip, you can compress the result further.

Q 1.17: find . -name "COMP*" -print: the find command takes the regular expression pattern in double quotes "...", where the quote is important to have.

Q 1.19:  correct answer should be: ln -s names snames (symplic link "target" is the second argument).
