COMP2011 Lab 7: Bubble Sort, Again!


Introduction

In programming assignment 1, you have implemented Bubble Sort. As mentioned, Bubble Sort is a well-known sorting algorithm (you can find detailed information here, and visualization here).

In this lab, you are going to implement the bubble sort algorithms on two types of data types: double array, and cstring array. Given two types of arrays, your task is to sort them using the functions defined in the utils.cpp, and you are not allowed to modify the contents in the main.cpp.

Tasks

You will implement two overloaded bubbleSort functions, and then conduct separate compilations.

Task1

Implement the bubbleSort function in utils.cpp, which can be used for sorting double floating point type arrays. In the main function, you will be asked to input the array size first, and then you are asked to input the content of the array. The contents are floating point numbers. The array and its size will be passed to this function.

This function also contains a default argument, ascending, which is set to be the value, true, by default, meaning sorting in the ascending order. By setting this argument to false, the array is sorted in descending order. You also need to implement this default argument in your code.

Task2

Implement the bubbleSort function in utils.cpp, which can be used for sorting cstring arrays. In the main function, you will be asked to input the array size first, and then you are asked to input the content of the array. The contents are cstrings in this task. The array and it size will be passed to this function.

The function also contains a default argument, ascending, which is set to be true by default, meaning sorting in the acsending order. By setting this argument to be 0, the array is sorted in descending order. You also need to implement this default argument in your code.

In sorting the cstrings, you may need to use the strcmp and strcpy functions for this task.

Task3

Write your function prototypes in my_include.h, and then conduct the separate compilation. The main function is defined in main.cpp. Note that modifying the main function is not allowed in this lab.


Implementation

  • Start your coding with the package. It contains skeleton codes, i.e. 3 files, namely, main.cpp, my_include.h, utils.cpp.
  • You only need to implement the functions in utils.cpp.
  • You don't need to change the function headers.
  • You can compile the program with the following command:
    For Win, 
    g++ -o lab7.exe main.cpp utils.cpp
    For Mac, 
    g++ -o lab7.out main.cpp utils.cpp
