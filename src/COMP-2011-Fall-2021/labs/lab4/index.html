COMP2011 Lab 4


Introduction

In this lab, you will practice array definition, initialization and operations.

A skeleton code has been provided to help you get started. Feel free to start from scratch without the use of skeleton if you wish, but maintain the given sample arrays for demo.

Tasks

The tasks in lab 4 is to implement several flipping functions which deal with 2D image (saved in 2D character array).

More concretely, there are 4 tasks in total to be implemented, each one is corresponding to a function to be filled in:

  • Task 1. Implement the function h_flip:
    Horizontal flip of the 2D array, concatenate with the original 2D array.
    Given a 2D character array, you can imagine there's a y-axis at its rightmost.
    The task is to flip the array horizontally along this y-axis, then output a larger array, which is the concatenation of the original array together with the flipped one.

  • Task 2. Implement the function v_flip:
    Vertial flip of the 2D array, concatenate with the original 2D array.
    Given a 2D character array, you can imagine there's an x-axis beneath its last row.
    The task is to flip the array vertically along this x-axis, then output a larger array, which is the concatenation of the original array together with the flipped one.

  • Task 3. Implement the function hv_flip:
    Horizontal and vertical flips of the 2D array, concatenate with the original 2D array.
    Given a 2D character array, you can imagine there's a y-axis at its rightmost, and an x-axis beneath its last row.
    The task is to flip the array horizontally and vertically so that original array together with its flipped versions, can fill up all the four quadrants.

  • Task 4. Implement the function initialize_self_defined_2D_array:
    For the above tasks, we've provided you 3 examples. To practice 2D character array initialization, this task asks you to self-define an array, and calculate some essential values which will be needed to call the above-mentioned flip functions. The array self-definition process should not be hard coded. When you run the code in the terminal, the 4th choice will be "Any Self-defined Array". If you choose this option, then you will need to input the number of rows of your array, then type in your array line by line in the terminal. This function will read your input and then initialize the array.

Grading Scheme

Each lab carries 3 points in total.

  • By attending a lab, a student gets 1 point for attendnace (but only if it is clear that the student does put in some effort).
  • By completing the lab exercise correctly and answering TAs' question (if there is a question) correctly, the student gets the remaining 2 more points.
If a student is unable to finish the lab and demonstrate to the TA by the end of the lab, they can only get the attendance point.
