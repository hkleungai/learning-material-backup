COMP2011 Lab 6: Zombie Attack


Introduction

In this lab, you are going to implement an algorithm according to the Zombie Attack game with recursion. Given the initial states of the farm and the location of zombie, your task is to report all the states of the farm in the next few timestamps.

Description

A zombie shows up in your farm and start to infect your plants. You are given the initial state of the farm and a timestamp (i.e. depth of recursion). To better defend the zombies, it would be great if you can tell the state of the entire farm in the next few timestamps. Could you help to do that?
Some useful assumptions:
  • The farm is a MxN grid, which has been pre-defined in the test cases.
  • Once the zombie location (x,y) is defined, cell (x,y) would turn into a zombie at T[0] (T[i] means timestamp `i` or when the recursion depth is `i` ).
  • In each timestamp, the zombie will keep infecting the plants nearby, and the infection is four-way(up, down, left, right).
  • The sunflower will turn into a zombie at T[i+1] if it get infected at T[i].
  • The nut will become a damaged nut at T[i+1] if it get infected at T[i].
  • The damaged nut won't turn into a zombie, and it will remain as damaged nut till the end of the game.
Below is a visual demonstration of the game.

Implementation

Start your coding with skeleton code and complete the core functions. Here are some pre-defined markers used in the skeleton code.
  • 'x': a cell with a zombie
  • '.': a cell with a sunflower
  • '#': a cell with a nut
  • '+': a cell with a damaged nut
Here are some rules and hints you need to remember.
  • Rule : You have to use recursion function to implement the the core function infect_farm.
  • You don't need to change the including headers and the provided helper functions.
  • Hint1 : You can add helper functions (which are not necessarily to be recursive)
  • Hint2 : There are some suggested sub tasks to be implemented in the helper functions (is_zombie and infect_cell). These two helper functions are optional for you to implement, but they may help divide and conquer the problem.
  • Hint3 : Try to be very careful with the boundary conditions. For example, when the initial location of zombie is a cell with nut and when a nut is sorrounded by more than one zombies.

Grading Scheme

Each lab carries 3 points in total.

  • By attending a lab, the student gets 1 point for attendance (but only if it is clear that the student does put in some effort).
  • By completing the lab exercise correctly and answering TAs' question (if there is a question) correctly, the student gets the remaining 2 more points.
  • You are recommended to turn on the camera while demonstrating.
If the student is unable to finish the lab and demonstrate to the TA by the end of the lab, they can only get the attendance point.
