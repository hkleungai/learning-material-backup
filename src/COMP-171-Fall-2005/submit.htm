How to submit the assignment?

The programming assignment 1 should be submitted by October 11th, 2005 (at 23:59:59) which is 3 weeks after the assignment released. Note that you need to produce ONE ZIPPED FILE FOR EACH PROBLEM. Each zip must be named according to the naming convention, "pa1A.zip" for problem A and "pa1B.zip" for problem B.

To submit your assignment, you should upload your ZIP files to the Course Assignment Submission System (CASS) before the deadline.

The assignment name for this programming assignment is "COMP171 PA1 (L#)" where # is the lecture section number. It is important that you choose the correct lecture section when you upload the ZIP files. Otherwise we may not able to locate your files.
