HTML Tutorial

The following presentation explains some basics of HTML so that you can understand it. This is an overview rather than an in-depth HTML reference.
  • Netscape "Composer" and various word processors (such as Microsoft Word) can convert formatted text documents to HTML, so normally you don't need to work directly in HTML
  • But you can modify a Web page using HTML commands that let you refine the formatting more precisely than you can with Composer.

HTML

  • stands for Hypertext Markup Language
  • is the programming language used in Web pages
  • files are normal ASCII text

Page Source

  • View —> Page Source will show you the HTML code for the current page
  • Netscape uses special colors to show:
    • HTML instructions — >purple
    • HTML instruction attributes — black bold
    • Data assigned to attributes — blue
    • Web page text in normal black letters
    • Comments in italics

HTML Tags

  • tell the Web browser which parts of the file are text or images, and which are programming code
  • begin with a "<" and end with ">"
  • usually come in pairs
    • <TITLE>Title Between Title Tags</TITLE>
    • The ending tag includes a slash "/" after the first bracket
  • are easier to see if they are in ALL CAPS
    • Blank Web Page
    <HTML>

    <HEAD>
    </HEAD>

    <BODY>
    </BODY>

    </HTML>
    • begin and end a Web document to tell the browser to read it as an HTML document
      • <HTML>Document Between HTML Tags</HTML>
    • begin and end each of the two parts of the Web document
      • <HEAD>Head of Document Between HTML Tags</HEAD>
      • <BODY>Body of Document Between HTML Tags</BODY>
  • the <BR> tag is a single tag for a line break

The Head Section

  • If a normal ASCII editor (such as Wordpad) created the Web page, the Head section usually just contains the title of the page (example: "Vina's Web Page")
  • If a word processor created the Web page, the Head section also contains a line showing the name of the word processor
The Head Section
<HTML>

<HEAD>
<META NAME="Generator" CONTENT="Microsoft Word 97">
<TITLE>Vina's Web Page </TITLE>
</HEAD>

<BODY>
</BODY>

</HTML>

The Body Section

  • Everything else on the page goes in the body
  • For example, let's add a title to the page
    • NOTE: there are two kinds of titles:
      • the one in the Head appears in the title bar
      • the one in the Body appears at the beginning of the page
    • Let's center the title within the <P> and </P> tags, which set it off as a paragraph
The Body Section
<HTML>

<HEAD>
<TITLE>Vina's Web Page </TITLE>
</HEAD>

<BODY>
<P ALIGN="CENTER">Vina's Home Page </P>
</BODY>

</HTML>

Tables

  • Let's add a table to show Vina's weekly activities.
  • Tables begin with the <TABLE> tag and end with the </TABLE> tag.
    • If you want a border, begin with the <TABLE BORDER> tag.
  • Rows begin with the <TR> tag and end with the </TR> tag.
  • Data cells begin with the <TD> tag and end with the </TD> tag.
Tables
<TABLE BORDER>
<TR>
<TD>
Sunday
</TD>
<TD>
<...>
</TD>
</TR>
</TABLE>
  • Table header cells begin with the <TH> tag and end with the </TH> tag.
  • To put a header (or data) cell across several other cells, use COLSPAN
    • Example: <TH COLSPAN=7>
Header Cells
<TABLE BORDER>
<TR>
<TH COLSPAN=7>
<CENTER>My Weekly Schedule</CENTER>
</TH>
</TR>
<TR>
<TD>
<...>
</TD>
</TR>
</TABLE>
