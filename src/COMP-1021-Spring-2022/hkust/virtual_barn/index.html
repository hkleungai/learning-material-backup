COMP1021 The Virtual Barn

  • The Virtual Barn is a service provided by HKUST which gives you access to Microsoft Windows through the web
    • It is available on Windows, Mac OS, Android and Apple iOS
    • So you can run it on your laptop, tablet, and even your phone
  • Sometimes it is very useful, so you are strongly recommended to install it on your machine
  • Here are some ways in which it is useful:
    • It can give you a path to HKUST through the web which works better for some users e.g. mainland students
    • It can give you access to Windows which already has Python installed, which you can use
      • As well as Python, you can use browsers and Microsoft Office programs such as Word and Excel
      • Because the Virtual Barn is 'part of HKUST' after you get into the virtual barn you don't have to worry about HKUST authentication again
  • To use the Virtual Barn, you need to install a program VMware Horizon Client on your device
  • Then you use the program to connect to the Virtual Barn on your device, then you can use the Virtual Barn
